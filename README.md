 -  [x] 多文件处理
 -  [x] 图形化界面
 -  [x] 纯本地运行，无需担心数据被窃取
 -  [ ] 原本想要实现多线程加速的，但是不知道为什么使用多线程之后程序不稳定，所以我就没做多线程了
 -  [ ] 比起上面其实还有一个更加激进的想法，调用显卡来加速处理，但是连上面的那个我都没搞定，这个就先不考虑了
